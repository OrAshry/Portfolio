Portfolio Website - Or Ashry
This repository contains the source code for my professional portfolio website, hosted at orashry.com. The site serves as a central hub to showcase my background as a Computer Science student, my technical skills, and the projects I have developed.

Overview:
The website is designed with a focus on clean system logic and a professional aesthetic, reflecting my interest in backend development and cybersecurity. It is fully responsive, ensuring a smooth experience across devices from mobile phones to large desktops.

Key Features:
Professional Summary: Highlights my academic focus at the Open University of Israel and my passion for low-level programming.

Interactive Skills Showcase: Displays technical proficiency through a categorized skills table and visual progress bars for programming languages.

Project Gallery: Direct links to my major GitHub projects, including a Python-based Work Tracker and a C-based Two-Pass Assembler.

Integrated CV: A direct download link for my latest professional resume.

Contact System: A functional contact form integrated with Web3Forms for direct inquiries.

Technologies Used:
HTML5: For semantic structure and search engine optimization (SEO).

CSS3: Custom styling using Flexbox and Media Queries for a fully responsive "dark mode" design.

JavaScript: Used for interactive elements and user alerts.

Font Awesome: For professional iconography.

Google Fonts: Specifically utilizing 'Poppins' for clean typography.

Project Structure:
index.html: The primary landing page featuring the intro, skills, and projects.

about.html: A detailed biography and personal mission statement.

styles.css: Central stylesheet containing all responsive layout rules.

assets/: Directory containing the project banner, profile images, and the downloadable CV.
