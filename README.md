# Portfolio Website

**Live Site:** [orashry.com](https://orashry.com)

**My Portfolio Website** is a responsive personal hub designed to showcase my academic background, technical skills, and software projects. It serves as a central professional presence for recruiters and collaborators to explore my work.

Building this site was my entry point into web development, where I learned HTML, CSS, and JavaScript from scratch to create a platform that reflects my passion for system logic and clean design.

---

## Background

Building this site was a practical experiment in web development. I created it to provide a more comprehensive and engaging view of my accolades than a standard resume allows, while simultaneously learning a new set of tools (HTML, CSS, and JavaScript) to see if I enjoyed full-stack development.

I believe you can't truly know if you like a field until you try it; through this project, I discovered that my passion remains firmly with low-level systems and cybersecurity rather than web development. This site serves as a central hub that gives recruiters a direct look at my technical discipline and the self-study approach required by my degree program at the Open University of Israel.

---

## Features

* **Responsive Design**: A fully optimized layout that provides a seamless experience across mobile, tablet, and desktop devices.
* **Skills Dashboard**: Visual progress bars and tables highlighting proficiency in C, Java, Python, and web technologies.
* **Project Gallery**: Integrated links to my GitHub repositories, showcasing practical applications of my programming knowledge.
* **Contact Integration**: A functional contact form using Web3Forms, allowing for direct professional inquiries.
* **Professional CV Access**: A dedicated download feature for my latest resume to streamline the hiring process.

---

## Tech Stack

* **Languages**: HTML5, CSS3, JavaScript
* **Styling**: Custom CSS with Flexbox and Media Queries
* **Icons**: Font Awesome
* **Typography**: Google Fonts (Poppins)

## Lessons Learned

Building this portfolio taught me:
* **Web Development Foundations**: Learning HTML, CSS, and JavaScript from the ground up to build a functional site.
* **Responsive Web Design**: Implementing complex CSS rules to handle various screen resolutions and orientations.
* **SEO & Web Optimization**: Utilizing meta tags and preloading assets to improve search engine visibility and performance.
* **Version Control**: Managing the deployment and updates of a live site using Git.
* **Career Clarity**: Gained a definitive understanding of my technical preferences by testing a new domain, reaffirming my commitment to cybersecurity, low-level programming, and system logic.
